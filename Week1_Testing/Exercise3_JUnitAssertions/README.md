# Exercise 3: Assertions in JUnit

A standalone test case to practice basic assertion checks from the core JUnit framework.

## Key Concepts Covered:

- Equality Verification
  Uses `assertEquals` to validate that math constants match.

- Boolean Conditions
  Evaluates logical operators using `assertTrue` and `assertFalse`.

- Reference Testing
  Checks for object identity and existence using `assertNull` and `assertNotNull`.
