# Cognizant Digital Nurture 5.0 - Git Hands-On Labs (HOL 1 to 5)

This directory contains automated scripts and artifacts for Git Hands-On Labs 1 to 5.

## Lab Summary
- **HOL 1: Setup & Basics** - Repository initialization (`GitDemo`), global configuration, adding `welcome.txt`, committing, and pulling/pushing.
- **HOL 2: Git Ignore** - Configuring `.gitignore` to exclude `.log` extension files and the `log/` folder.
- **HOL 3: Branching & Merging** - Creating `GitNewBranch`, switching branches, checking differences, merging into `main`/`master`, and deleting the branch.
- **HOL 4: Conflict Resolution** - Creating `GitWork` branch with `hello.xml`, introducing conflicting changes on `main`/`master`, resolving merge conflicts manually, updating `.gitignore`, and branch cleanup.
- **HOL 5: Remote Sync & Cleanup** - Checking branch statuses, pulling remote updates, pushing local master updates, and verifying clean tree state.
