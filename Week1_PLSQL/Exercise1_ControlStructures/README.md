# Control Structures Notes

This part is all about setting up the database tables and writing some basic bank logic using PL/SQL.

## What we built:

- Tables & Data: Created the main Customers and Loans tables and tossed in some test data.
- Senior Discount: A check that looks at a customer's age. If they are over 60, it automatically lowers their interest rate by 1%.
- VIP Status: A system that checks for accounts with a balance over 10,000. If it finds one, it upgrades them to VIP status.
- Loan Alerts: A simple alert that pops up on the screen to remind people if they have a loan due in the next 30 days.
