## Setup & Execution

1. Clone this repository to your local machine:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the specific week and assignment directory.
3. Import the project as a **Maven Project** into your preferred IDE (Eclipse, IntelliJ IDEA, STS).
4. For backend modules, update the `application.properties` or `application.yml` file with your local database credentials.
5. Build the project using Maven:
   ```bash
   mvn clean install
   ```
6. Run the application via the main Spring Boot class or execute specific JUnit test files.
